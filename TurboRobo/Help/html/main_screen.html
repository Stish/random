Actions

Forward, right

Backward, right

Help

Shows short information about the selected action/sensor/logic symbol. It describes how to use the parameters and how to call the functions directly in C source code.


Parameters

The parameters for the selected panel can be directly changed in the parameter window. With clicking on ‘Apply’ the changes will be applied. ‘Reset’ causes a reset of the parameters as well as a reset of the panel itself.


Program Panels



Sequences


Actions can be easily dropped into the sequence panels. Sequences can be called from the main program or from the sensors. Use sequences to generate motion patterns, which can be called from one panel.


Main Program


This is the main program, which is running, if no sensor is triggered. It can be looped with a right click on the blue M panel in the header of the main program line.


Sensors


Sensor icons can be easily dropped in the sensor header panels. If the sensor is triggered, the main program is interrupted and the actions on the corresponding sensor line are executed. After finishing the actions, the main program is restarted. If the main program is not looped, then no actions will take place.


Status Bar



The status bar shows the position of the selected panel.

Columns 1-3: Sequences

Column 4: Main Program

Columns 5-10: Sensors


The debugger information is displayed as well. If an error occurs, it’s not possible to generate C source code out of the project.



Source Code



After finishing the project, it’s necessary to generate C source code out of it. The generated code is displayed in the memo window. The source code can be compiled with ‘make all’ from the Run menu and than uploaded to the micro controller via MegaLoad.
