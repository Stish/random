Program List

Shows the list of actions, which will be performed in the program. The action is named and displayed with the duration time. You can select from which action you want to simulate. The actions from sequence calls are directly included. In this version of the simulator, all actions will be performed with speed 10, no matter what speed was chosen in the program. Speed adjustment will be included later.

Manual Actions

Use the manual action window to see the motion of the robot for a defined time. Enter the time into the edit window and choose an action with clicking on it. The robot will perform the motion. In this version of the simulator, all actions will be performed with speed 10. Speed adjustment will be included later.


Simulate Screen



The simulate screen shows the action, the robot will perform. The track can be displayed as well. It simulates the motion in a room for better time adjustment of your project.


Status Bar



The status bar is divided into ‘Time’ and ‘Time all’. ‘Time’ shows the time passed for each action and ‘Time all’ for the whole program in seconds. ‘Time’ is set to zero, if a new action starts.
